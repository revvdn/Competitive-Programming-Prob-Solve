# Graph Theory

Graph theory deals with nodes (vertices) and connections (edges).

## Key Patterns
- **Traversal**: DFS and BFS.
- **Shortest Path**: Dijkstra, Bellman-Ford, Floyd-Warshall.
- **Minimum Spanning Tree**: Kruskal's, Prim's.
- **Connectivity**: DSU (Disjoint Set Union).

## Common Techniques
- **Adjacency List**: Efficient way to store graphs.
- **Binary Lifting**: For LCA or reachability in functional graphs.

## When to Use
- Relationship modeling.
- Pathfinding.
- Network flows.
