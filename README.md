# Competitive Programming Problem Solving

A structured repository containing solutions to various competitive programming problems from platforms like Codeforces, CSES, and AtCoder.

## Platforms Covered
- **Codeforces**: Round solutions and gym problems.
- **CSES**: Problems from the CSES Problem Set.
- **AtCoder**: Beginner and Regular contest solutions.

## Repository Structure
- `/problems/`: Categorized solutions by platform and topic/contest.
- `/templates/`: Reusable C++ templates for common algorithms (DSU, Segment Tree, etc.).
- `/notes/`: Technical notes on key algorithmic patterns.

## Features
- Standardized headers with problem metadata.
- Automated problem indexing.
- Clean and readable C++ implementations.
